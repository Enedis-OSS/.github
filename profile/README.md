# ⚡ Enedis Open Source Software (Enedis-OSS)

⭐ Welcome!

Enedis promotes **Open Source technologies** to facilitate knowledge sharing, foster collaboration, and accelerate innovation with partners in the **energy and utilities sector**.

This GitHub organization hosts **open-source projects, tools, and contributions** developed by Enedis, a major French **electricity distribution system operator (DSO)** and a key player in **smart grid ecosystems**.

---

## 🎯 Motivation

### ✅ An Open Source–Ready Policy

Since **March 2016**, the Enedis IT Department has actively embraced an **Open Source strategy**, promoting adoption and contribution within the company.

In **2021**, this commitment was reinforced with an **Open Source Ready policy**, supported by a dedicated governance structure to:

- Encourage the use of **open technologies**
- Ensure compliance with **open-source licenses**
- Promote internal and external **collaboration**
- Structure contributions to the **open-source ecosystem**

---

### 🌍 Enedis & the Open Source Community

Enedis actively contributes to the open-source ecosystem and is a founding member of:

- [**TOSIT The Open Source I Trust)**](https://tosit.fr/),
  → Promotes enterprise-grade open-source solutions, especially in **cybersecurity**

- [**Hermine**](https://gitlab.com/hermine-project/hermine), 
  → A tool dedicated to **open-source license compliance**

👉 You can also explore related initiatives on the [**EDF GitHub organisation!**](https://github.com/groupe-edf).

## 👊🏻 Our Projects

Our projects focus on **energy systems, smart grids, data engineering, and software tooling**.

### 🔧 Selected Projects

- [**mocha-xray-reporter**](https://github.com/Enedis-OSS/mocha-xray-reporter),
  → Generates Xray-compatible reports from Cypress tests, including automated screenshots for **acceptance testing**

- [**chutney**](https://github.com/Enedis-OSS/chutney), 
  → A flexible **integration and end-to-end testing framework** using a Kotlin DSL

- [**grafana-dsl**](https://github.com/Enedis-OSS/grafana-dsl),
  → A Kotlin-based DSL for defining **Grafana dashboards as code**, improving observability management

- [**TIC2Zigbee**](https://github.com/Enedis-OSS/TIC2Zigbee), 
  → Defines Zigbee clusters and mappings for the **Linky TIC (Télé Information Client)** interface  
  → Supports interoperability in **smart meter ecosystems**

- [**TIC2WebSocket**](https://github.com/Enedis-OSS/TIC2WebSocket), 
  → WebSocket API to access and stream **real-time Linky TIC data**

- [**tic4eebus**](https://github.com/Enedis-OSS/tic4eebus), 
  → Energy management use case integrating **EEBUS protocol** and Linky TIC  
  → Enables **smart EV charging based on available energy**

- [**StaREngine**](https://github.com/Enedis-OSS/StaREngine),  
  → Engine for processing **electrical network data (GML)**  
  → Ensures compliance with **RecoStaR / StaR-Elec standards**
